About coach-in

Coach-inは電気刺激を用いて装着者の身体を駆動させるウェアラブルデバイスです。 スポーツトレーニングのコーチが身体の動かし方を教えてくれるように、ウェアラブルデバイスがあなたの身体を駆動させ教示することを目指しています。 自分の身体が誰かに操られる、そんな体験を提供します。

Technology

EMS

Electric Muscle Stimulationによる身体駆動

筋肉は、脳からの命令となる電気的な刺激を受け運動します。 電気筋肉刺激（Electric Muscle Stimulation：EMS）は脳の命令と同様の刺激を電気によって与え筋肉を運動させる技術です。Coach-inはこの技術を用いユーザの指を動かします。我々は指の駆動にEMSを用いることで、モータやワイヤを用いる場合と比べ軽量・小型なウェアラブルデバイスを開発しています。
#stimulation #muscle

iOS Application

Bluetoothによる様々な機器との連携

Coach-inにはBluetoothモジュールが搭載されており、コンピュータやスマートフォンと通信することで様々な機器と連携することができます。 例えば、コンピュータに接続されたゲームコントローラーで他人の身体を遠隔操作したり、スマホで再生している音楽に合わせて勝手に足踏みしてしまう体験を提供できます。
#swift #bluetooth

Sensor

センサフュージョンによる行動センシング

Coach-inには加速度センサ・ジャイロセンサ・筋電位センサが搭載されており、ユーザの行動をリアルタイムにセンシング・分析します。 Coach-inはこれらのセンサのデータをリアルタイムに分析することで、あなたの身体へ適切な筋肉の動かし方を指示します。
#ESP32 #bluetooth

Virtual Assistant

音声対話インターフェイスによる情報入力

Coach-inに搭載されたマイクはあなたと音声対話エージェントとの会話を可能にします。 マイクから入力された音声はクラウドサーバで音声解析され、その結果はCoach-inに取り付けられたLEDの光とスピーカーからの音声でユーザに伝わります。 我々は自分とは異なる存在が身体にインストールされる体験を提案します。
#alexa #aws lambda
